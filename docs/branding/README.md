# NDB-TV Brand Guidelines

## Purpose

Consistent visual identity across all platforms and content.

---

## Logo

### Primary Logo
- **File:** `assets/logo/ndb-tv-logo-primary.png`
- **Usage:** Channel icon, video intro, website header
- **Clear space:** Minimum 1x logo height on all sides

### Secondary Logo (Icon-only)
- **File:** `assets/logo/ndb-tv-logo-icon.png`
- **Usage:** Social media avatar, app icon, small spaces

### Logo Do's
- Use on clean backgrounds
- Maintain aspect ratio
- Use provided color versions only

### Logo Don'ts
- Don't stretch or distort
- Don't change colors
- Don't add effects (shadows, outlines)
- Don't place on busy backgrounds without sufficient contrast

## Color Palette

| Color | Hex Code | Usage |
|-------|----------|-------|
| Primary Blue | #1a73e8 | Headers, buttons, links |
| Dark Blue | #0d47a1 | Backgrounds, emphasis |
| White | #ffffff | Text on dark backgrounds |
| Light Gray | #f5f5f5 | Background sections |
| Accent Orange | #ff6d00 | CTAs, highlights |
| Accent Green | #00c853 | Health content, positive messages |

---

## Typography

### Primary Font (Arabic)
- **Noto Sans Arabic** (Google Fonts, free)
- Usage: Titles, body text, subtitles

### Secondary Font (English)
- **Inter** (Google Fonts, free)
- Usage: English subtitles, technical content

### Type Scale

| Usage | Size | Weight |
|-------|------|--------|
| Video titles | 48–64px | Bold |
| Section headers | 32–40px | Semi-bold |
| Body text | 16–20px | Regular |
| Captions | 14–16px | Regular |
| Lower thirds | 24–32px | Bold |

---

## Thumbnail Design System

### Elements
1. **Subject** — Face visible, expressive, well-lit
2. **Background** — Clean, contrasting, branded color or scene
3. **Text** — 3–5 words maximum, bold, high contrast
4. **Logo** — Small, bottom-right corner

### Style Rules
- Consistent border/frame style
- Font: Noto Sans Arabic Bold
- No more than 1/3 of frame covered by text
- Text always readable at mobile size (320px width)

---

## Intro/Outro

### Intro Animation
- Duration: 5 seconds
- Content: Logo animation + channel name + tagline
- Audio: Consistent brand sound/music

### Outro End Screen
- Duration: 10 seconds
- Elements: Logo, 2 recommended videos, subscribe button
- Consistent layout across all videos

---

## Asset File Naming

Format: `ndb-tv_[asset-type]_[descriptor]_[size/format].[ext]`

Examples:
- `ndb-tv_logo_primary_1080p.png`
- `ndb-tv_banner_youtube_2560x1440.png`
- `ndb-tv_thumbnail_zidni-ai_001.jpg`

---

## Platform-Specific Guidelines

### YouTube
- Banner: 2560×1440px
- Profile photo: 800×800px
- Thumbnail: 1280×720px

### Instagram
- Profile: 320×320px
- Post: 1080×1080px
- Story: 1080×1920px
- Reel: 1080×1920px

### Facebook
- Cover: 1640×624px
- Profile: 180×180px
- Video: 1280×720px

### TikTok
- Profile: 200×200px
- Video: 1080×1920px
