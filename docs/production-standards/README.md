# Production Standards

## Purpose

Consistent quality across all NDB-TV content. These standards apply to every episode, short, livestream, and published piece.

---

## Video Production Standards

### Technical Requirements

| Element | Minimum | Recommended |
|---------|---------|-------------|
| Resolution | 1080p | 4K |
| Frame rate | 24 fps | 30 fps |
| Audio quality | Clear, no background noise | Dedicated microphone |
| Lighting | Subject clearly visible | 3-point lighting |
| Aspect ratio | 16:9 (horizontal), 9:16 (shorts) | — |

### Audio Standards
- Voice must be clear and intelligible above background
- Background music must not overpower speech
- No abrupt volume changes
- Intro/outro audio consistent across episodes

### Visual Standards
- Text overlays: Readable font, sufficient contrast, appropriate size
- Lower thirds: Consistent design, correct spelling of names/titles
- B-roll: Relevant, properly licensed or original footage
- Color grading: Consistent tone throughout an episode

---

## Episode Structure

### Standard Episode (5–15 min)
1. **Hook** (0–15s) — Grab attention, state the value
2. **Intro** (15–60s) — Host introduction, topic framing
3. **Content** — Main educational/informational content
4. **Summary** — Key takeaways recap
5. **Call to Action** — Subscribe, comment, next episode
6. **Outro** — Channel branding, end screen

### Shorts (30–60s)
1. **Immediate hook** — First 2 seconds
2. **One idea** — Single clear message
3. **Visual focus** — Tight framing, text overlay
4. **Quick CTA** — Last 3 seconds

## Publishing Checklist

Before every publish:
- [ ] Title optimized (descriptive, keyword-rich, Arabic first)
- [ ] Thumbnail designed (high contrast, face visible, text readable)
- [ ] Description written (summary, relevant links, timestamps)
- [ ] Tags added (Arabic keywords)
- [ ] Chapters added (for 5+ min videos)
- [ ] Subtitles uploaded (Arabic, English for AI content)
- [ ] End screen elements set
- [ ] Social cross-post prepared

---

## Format Standards

### Titles
- Arabic primary (Hassaniya/Darija for conversational content)
- English subtitle in parentheses for AI content
- Maximum 60 characters
- No clickbait or misleading titles

### Descriptions
- Summary paragraph (2–3 sentences)
- Timestamps for long videos
- Links to resources mentioned
- Social media links
- Sponsorship disclosure (if applicable)
- Health disclaimer (for health content)

### Thumbnails
- High contrast, readable at small size
- Face visible, expressive
- Minimal text (3–5 words max)
- NDB-TV consistent color palette
- No misleading imagery
