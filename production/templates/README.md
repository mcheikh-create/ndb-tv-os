# NDB-TV Templates — Production Directory

## Purpose
Reusable templates for consistent video production across all NDB-TV content formats.

## Structure

| Directory | Content |
|-----------|---------|
| `youtube/` | Long-form video templates, intro/outro layouts, end screen configurations |
| `shorts/` | YouTube Shorts templates — vertical 9:16 format |
| `reels/` | Instagram/Facebook Reel templates |
| `thumbnails/` | Thumbnail design templates with NDB-TV branding |
| `lower-thirds/` | Lower-third graphic templates for presenter/topic identification |
| `episode-packages/` | Complete episode packages combining multiple templates |

## Design Principles
- All templates use NDB-TV brand colors: deep blue (#0a1f3d), white, red (#e63946)
- Logo placement follows brand guidelines
- Templates are reusable — fill in content, export, publish
- Source files (PSD, AI, SVG) kept alongside production PNG/MP4 files
