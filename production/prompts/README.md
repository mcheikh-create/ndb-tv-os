# NDB-TV Prompts — Production Directory

## Purpose
Reusable prompt templates for AI-assisted content creation: avatar video generation, script drafting, translation, and localization.

## Directory Contents

| File | Purpose |
|------|---------|
| `avatar-video-prompts.md` | Prompts for generating avatar presenter videos |
| `presenter-script-prompts.md` | Prompts for drafting human presenter scripts |
| `education-content-prompts.md` | Prompts for educational explainer content |
| `translation-localization-prompts.md` | Prompts for translating and localizing content |

## Usage Rules
1. All AI-generated output must be reviewed by a human before use.
2. Do not use prompts to generate unverified factual claims.
3. Attribution: content drafted by AI, verified by NDB-TV editorial team.
4. Prompts are starting points — always customize for specific episodes.

No prompts have been written yet. Directory prepared for future development.
