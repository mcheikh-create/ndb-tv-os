## Purpose

This package contains the complete thumbnail design specification for NDB-TV's first public video. The thumbnail is the first visual impression viewers have of NDB-TV — it must be professional, local, simple, and consistent with Brand v1.0.

## Design Philosophy

- **Simple** — Not crowded. One clear focal point.
- **Local** — Founder photo + Arabic title. Nouadhibou identity.
- **Professional** — Clean layout, brand colors, no clip art.
- **Non-hype** — No exclamation marks, no "WOW" language, no exaggerated expressions.

## Required Founder Approval

Before the thumbnail is uploaded to YouTube, the founder must approve:

1. Title text selection (from `text-options.md`)
2. Layout option (from `layout-guide.md`)
3. Founder photo (real photo provided by founder — not AI-generated)
4. Final rendered PNG
