# Livestream Operations

## Purpose

Weekly livestream is the anchor event of NDB-TV's publishing calendar. It builds community, creates real-time engagement, and generates content for shorts.

## Livestream Types

### 1. Talk Show
- Host discusses topic(s) of the week
- Live comments read and addressed
- Guest appearances (remote or in-studio)
- 45–60 minutes

### 2. Tutorial / Demo
- Screen share with live demonstration
- Real-time audience questions
- Step-by-step walkthrough
- 30–45 minutes

### 3. Community Chat
- Open mic format
- Community members share perspectives
- Moderated by host
- 45–60 minutes

### 4. Event Livestream
- Coverage of live events
- Multi-camera if available
- Pre-announced schedule
- Variable duration

## Pre-Production Checklist (72h before)

- [ ] Topic selected and announced
- [ ] Streaming platform confirmed (YouTube primary)
- [ ] Equipment tested
- [ ] Internet connection tested (minimum 5 Mbps upload)
- [ ] Backup plan prepared (prerecorded content if stream fails)
- [ ] Moderator assigned for chat
- [ ] Social media announcement posted
- [ ] Community reminder scheduled (24h before)

## Pre-Stream Checklist (1h before)

- [ ] Camera and lighting set
- [ ] Audio tested (levels, background noise)
- [ ] Streaming software configured (OBS or similar)
- [ ] Overlays/graphics loaded
- [ ] Test stream run (private, 30 seconds)
- [ ] Backup recording started (local)
- [ ] Chat moderators online
- [ ] Water/beverage ready for host

## During Stream

- Monitor chat for questions and engagement
- Read viewer names on air
- Keep segments to 10–15 minutes
- Take a brief mid-stream break if >45 min
- Moderator handles chat moderation
- Technical issues: explain, resolve, continue

## Post-Stream (within 24h)

- [ ] Stream made public (if not already)
- [ ] Stream trimmed (remove dead time at start/end)
- [ ] Timestamps added to description
- [ ] Key moments extracted for shorts (2–3 clips)
- [ ] Community post thanking viewers
- [ ] Analytics reviewed
- [ ] Notes for next stream

## Technical Requirements

### Minimum Setup
- 1080p webcam or phone camera
- USB microphone or headset
- Stable internet (5 Mbps upload)
- OBS Studio (free) or StreamYard

### Recommended Setup
- DSLR/mirrorless camera via HDMI capture
- Dedicated microphone (dynamic or condenser)
- Wired internet connection
- Secondary display for chat/monitoring

## Contingency Plans

| Failure | Response |
|---------|----------|
| Internet drops | Switch to phone hotspot, delay or cancel |
| Audio fails | Pause, fix, or switch to backup microphone |
| Camera fails | Continue with screen share/slides only |
| Software crash | Restart, apologize briefly, continue |
| Guest cannot connect | Host solo, reschedule guest |
