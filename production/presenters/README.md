## Core Principles

1. **Local voices matter.** Human presenters bring authentic lived experience and cultural connection that avatars cannot replicate.
2. **Professional but accessible.** Presenters represent the community — approachable, not distant.
3. **Consistent branding.** All presenters follow the same visual and delivery standards.
4. **Script approval.** All scripts reviewed by founder before recording.
