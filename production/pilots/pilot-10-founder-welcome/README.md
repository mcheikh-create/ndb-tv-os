## Purpose

This is the first public-facing NDB-TV video. Its purpose is to:

1. Introduce NDB-TV to the community in an authentic, personal way.
2. Explain what NDB-TV is and what it will cover.
3. Set expectations honestly — this is a new initiative being built.
4. Invite participation from teachers, youth, creators, families, and local voices.
5. Establish the tone: sincere, local, educational, community-building.

## Core Message

NDB-TV is being built as a media and learning network for Nouadhibou. It will support education, useful information, culture, youth learning, local stories, and community participation. The network will use human presenters and carefully labeled AI presenters where useful. Accuracy, respect, and community benefit are core rules.
