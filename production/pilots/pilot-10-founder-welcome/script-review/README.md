| File | Purpose |

## Critical Rule

> **No recording until the script is approved.**
>
> The script is a draft. Changes are expected and welcome. Once approved, everything is ready to proceed: the recording checklist, shot list, thumbnail, and private upload workflow are all prepared and waiting.

## How to Use This Packet

1. Read the script in `founder-review-version.md`
2. Read it aloud once (time yourself — target 4:30–5:30)
3. Fill in `script-approval-form.md` with your decision
4. If edits needed: fill in `edit-request-form.md`
5. Fill in `recording-decision-sheet.md` with your preferences
6. Return completed forms to Hermes (via this repo or direct instruction)
