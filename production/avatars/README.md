# NDB-TV Avatars — Production Directory

## Purpose
Contains all assets, profiles, voice guidelines, and style references for AI-generated avatar presenters used in NDB-TV content.

## Structure

| Path | Content |
|------|---------|
| `README.md` | This file — overview and rules |
| `presenter-profiles/` | Individual avatar character profiles (name, role, voice, appearance, personality) |
| `voice-guidelines/` | Voice model specifications, TTS configuration, pronunciation guides for Hassaniya/Arabic/French |
| `avatar-style-guide.md` | Unified visual and behavioral style guide for all NDB-TV avatars |

## Core Principles

1. **Transparency.** All avatar-generated content must be clearly labeled as "AI Presenter" in the video description and intro.
2. **Approval.** Every avatar and every script must be founder-approved before production.
3. **Consistency.** Avatars maintain consistent appearance, voice, personality, and on-screen positioning across all episodes.
4. **No impersonation.** Avatars must not resemble any real person without explicit permission. They are clearly synthetic.
5. **Educational first.** Avatars are primarily used for educational, informational, and public-service content — not opinion or editorial.

## Avatar Categories

| Category | Tone | Best For |
|----------|------|----------|
| Educator | Warm, patient, clear | Tutorials, explainers, how-to content |
| News Anchor | Professional, neutral, authoritative | News briefs, public announcements |
| Youth Host | Energetic, relatable, modern | Youth-focused content, skills training |
| Cultural Guide | Respectful, knowledgeable | History, culture, heritage stories |
