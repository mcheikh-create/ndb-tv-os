## If Script Is Approved (no edits or minor edits)

The following sequence activates immediately:

```
Step 1: Recording Preparation
  - Founder reads the recording checklist (founder-recording-checklist.md)
  - Equipment prepared: camera/phone, mic, tripod, lighting, quiet room
  - Teleprompter set up if requested
  - Target: one session, ~60 minutes

Step 2: Recording
  - Record 2 full takes of the approved script
  - Record closing section retake as backup
  - Save files with clear names: pilot10_take1.mp4, pilot10_take2.mp4
  - Backup files to second location before wrapping up

Step 3: Hermes Prepares Captions + Transcript
  - Founder shares the best take with Hermes
  - Hermes creates Arabic SRT captions
  - Hermes prepares video description text
  - Files saved to: production/pilots/pilot-10-founder-welcome/captions/

Step 4: Claude Chrome Private Upload
  - Uploads video as Private or Unlisted on YouTube
  - Attaches thumbnail (founder photo needed first)
  - Uploads Arabic captions
  - Shares private review link with founder

Step 5: Founder Review
  - Watches full video on YouTube (not local file)
  - Verifies captions, thumbnail, metadata
  - Signs final-publish-approval-sheet.md or requests changes

Step 6: Publish Decision
  - Founder approves public publish
  - Visibility changed from Private → Public
  - OR founder keeps video private and schedules future publish
```
