# NDB-TV Scripts — Production Directory

## Purpose
Repository of all written scripts organized by content category. Scripts are written, reviewed, and approved before any production begins.

## Categories

| Directory | Content Type | Format |
|-----------|-------------|--------|
| `educational/` | Tutorials, explainers, how-to guides | Script + source references |
| `news-briefs/` | Factual news summaries, public info | Script + source URLs |
| `culture/` | History, heritage, community stories | Script + research notes |
| `public-service/` | Health, safety, civic information | Script + authority sources |
| `templates/` | Reusable script frameworks | Fill-in-the-blank formats |

## Script Lifecycle

```
Draft → Founder/Editor Review → Approved → Record → Review → Publish
```

## Standards
- Every script must have a source/evidence checklist (see templates).
- Every script must pass founder/editor approval before recording.
- Scripts in Hassaniya Arabic preferred. MSA for formal content. French for bilingual content.
- English subtitles encouraged for international reach.
