# NDB-TV YouTube — Comment Moderation Policy

**Status:** Draft — awaiting founder approval
**Scope:** All NDB-TV YouTube channel comments

---

## 1. General Approach

Comments are **enabled** on all NDB-TV videos. YouTube's automatic moderation handles obvious spam. The founder or delegated moderator reviews comments manually in the early phase.

**Phase 1 (First 30 days / first 5 videos):**
- Comments published automatically after basic YouTube filter
- Founder reviews all comments daily
- Polite responses given by founder account where appropriate

**Phase 2 (After 30 days or 5+ videos):**
- If spam or abuse volume is low — maintain Phase 1
- If volume is high — hold all comments for review before publishing

---

## 2. Comments That Will Be Removed or Hidden

| Category | Example | Action |
|----------|---------|--------|
| Hate speech | Racist, ethnic, religious, or gender-based attacks | ❌ Remove immediately |
| Threats | Violence, harassment, doxxing | ❌ Remove + report to YouTube |
| Spam | Self-promotion links (not discussion), repeated links | ❌ Remove |
| Scams | Fake giveaways, phishing, impersonation | ❌ Remove + report |
| Personal attacks | Insults directed at the founder, presenters, or other commenters | ❌ Remove |
| Misinformation | Demonstrably false claims about health, science, or facts | ⚠️ Reply with correction + link. Remove if repeated. |
| Trolling | Bad-faith comments intended to provoke | ⚠️ Remove or hide |
| Off-topic promotion | Crypto, financial schemes, unrelated services | ❌ Remove |

---

## 3. Comments That Will Be Kept (Even If Critical)

| Category | Example | Action |
|----------|---------|--------|
| Respectful disagreement | "I don't agree with this approach because..." | ✅ Keep. Reply if useful. |
| Constructive criticism | "The audio could be clearer in the next video." | ✅ Keep. Thank the commenter. |
| Questions | "Can you make a video about X?" / "What does AI mean?" | ✅ Keep. Reply or use for future content. |
| Cultural/linguistic feedback | "This word in Hassaniya is said differently." | ✅ Keep. Thank for correction. |
| Suggestions | "I'd like to see a video about fishing industry." | ✅ Keep. Log for future content planning. |

---

## 4. Channel Account Rules

| Rule | Explanation |
|------|-------------|
| No public arguments | Never engage in back-and-forth arguments in comments. One polite reply, then stop. |
| No defensive tone | "Thank you for your feedback, we'll consider that." — never "You're wrong because..." |
| No political replies | If a comment brings up politics, do not engage. |
| No medical advice | If a comment asks for medical advice, reply: "We provide health education. Please consult a doctor for personal medical concerns." |
| No promises | "We'll try to cover that!" — not "We'll definitely make a video about that next week." |
| One account | All official replies come from the NDB-TV channel account only. |

---

## 5. Escalation Rules

| Situation | Action |
|-----------|--------|
| Comment contains threats | Do not reply. Screenshot. Report to YouTube immediately. |
| Comment appears to be from a real organization impersonating NDB-TV | Flag to founder. |
| Repeated harassment from one account | Block account after one warning in comments. |
| Legal concerns in a comment | Do not reply. Screenshot. Escalate to founder. |
| Comment goes viral on other platforms (negative) | Do not reply. Escalate to founder. |

---

## 6. Daily Moderation Workflow

1. Check YouTube Studio → Comments → "Published" tab — review all new comments
2. Check "Held for review" tab — approve appropriate comments
3. Check "Likely spam" tab — verify YouTube auto-filter is working
4. Reply to up to 3–5 comments per day (founder time permitting)
5. If any comment requires escalation → screenshot + message founder

**Estimated daily time:** 5–10 minutes
