# NDB-TV Pilot #10 — Post-Recording Edit Plan

**Status:** Pre-production — execute after recording
**Estimated edit time:** 2–4 hours
**Target format:** 1080p MP4, H.264, ~5 minutes

---

## Edit Workflow

### Step 1: Import & Review (20 min)

| # | Task | Done? |
|---|------|-------|
| 1.1 | Import all takes into editing software | ☐ |
| 1.2 | Review Takes 1 and 2 fully — note which take is best overall | ☐ |
| 1.3 | Identify sections from each take to combine (if needed) | ☐ |
| 1.4 | Mark the best take as primary timeline | ☐ |

### Step 2: Rough Cut (30 min)

| # | Task | Done? |
|---|------|-------|
| 2.1 | Place best full take on timeline | ☐ |
| 2.2 | Cut out stumbles, pauses, restarts | ☐ |
| 2.3 | If combining takes: find clean cut points at natural pauses | ☐ |
| 2.4 | Trim opening silence — video starts on "السلام عليكم" | ☐ |
| 2.5 | Trim ending — hold final frame 2 seconds, then fade to logo | ☐ |
| 2.6 | Rough timeline length: target 4:30–5:30 | ☐ |

### Step 3: Audio Cleanup (20 min)

| # | Task | Done? |
|---|------|-------|
| 3.1 | Normalize audio levels (voice at -6 to -3 dB peak) | ☐ |
| 3.2 | Remove background noise (noise gate or spectral cleanup) | ☐ |
| 3.3 | Gentle compression (2:1 ratio, -12 dB threshold) | ☐ |
| 3.4 | Ensure consistent volume throughout — no sudden jumps | ☐ |
| 3.5 | If using background music: add at -25 dB, very subtle | ☐ |

### Step 4: Visual Polish (30 min)

| # | Task | Done? |
|---|------|-------|
| 4.1 | **Intro:** Add NDB-TV logo opening card or intro animation (if available). Otherwise start cold with founder on camera. | ☐ |
| 4.2 | **Lower-third:** Add NDB-TV logo overlay (bottom-right) for entire video duration | ☐ |
| 4.3 | **Text overlay (optional):** Add program names during section 2 (زدني AI, أحب NDB, etc.) | ☐ |
| 4.4 | **B-roll (optional):** Insert at designated points if self-shot footage exists | ☐ |
| 4.5 | **Color grade:** Minor correction only. Warm, natural tones. No over-processing. | ☐ |
| 4.6 | **Outro:** Add end screen with NDB-TV logo + "شكرًا على المشاهدة — اشترك في القناة" | ☐ |

### Step 5: Subtitles & Captions (30 min)

| # | Task | Done? |
|---|------|-------|
| 5.1 | Create Arabic SRT file — match spoken Hassaniya exactly | ☐ |
| 5.2 | Sync subtitle timing to audio — check early/late appearance | ☐ |
| 5.3 | Export SRT file separately (YouTube accepts SRT upload) | ☐ |
| 5.4 | Optional: Create French SRT (summary, not literal) | ☐ |
| 5.5 | Verify subtitles readable — white text, black outline, proper size | ☐ |

### Step 6: Export (15 min)

| # | Task | Value |
|---|------|-------|
| 6.1 | Format | MP4 |
| 6.2 | Video codec | H.264 |
| 6.3 | Resolution | 1920×1080 |
| 6.4 | Frame rate | 30 fps |
| 6.5 | Bitrate | 8–12 Mbps (variable) |
| 6.6 | Audio codec | AAC |
| 6.7 | Audio bitrate | 192 kbps |
| 6.8 | File name | `pilot-10-welcome-v1.mp4` |

### Step 7: Save Project Files (5 min)

| # | Task | Done? |
|---|------|-------|
| 7.1 | Save editing project file (Premiere/DaVinci/Resolve/CapCut project) | ☐ |
| 7.2 | Export SRT files separately | ☐ |
| 7.3 | Save exported MP4 to `production/pilots/pilot-10-founder-welcome/` | ☐ |
| 7.4 | Note: Keep project file accessible for future edits | ☐ |

## Quality Checklist Before Upload

| # | Check | Pass? |
|---|-------|-------|
| A | Video length: 4:30–5:30 | ☐ |
| B | Audio clear — no hiss, no pops, consistent level | ☐ |
| C | No visible flubs or mistakes in final cut | ☐ |
| D | Lower-third logo present throughout | ☐ |
| E | Intro/outro clean and correct | ☐ |
| F | Subtitles uploaded as separate SRT file | ☐ |
| G | No political, exaggerated, or unverified content | ☐ |
| H | Tone matches script — sincere, local, educational | ☐ |
| I | Founder reviewed and approved final cut | ☐ |
| J | Thumbnail file ready (separate PNG) | ☐ |
