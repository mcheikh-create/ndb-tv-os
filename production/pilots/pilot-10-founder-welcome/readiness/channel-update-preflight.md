## Order of Operations

When the founder approves, update the channel in this order:

```
Step 1:      Profile picture + Banner           (YouTube Studio → Branding)
Step 2:      About section text                  (YouTube Studio → Customization → About)
Step 3:      Country, Language, Keywords         (YouTube Studio → Settings → Channel)
Step 4:      Links (WhatsApp, Email)             (YouTube Studio → About → Links)
Step 5:      Upload defaults                     (YouTube Studio → Settings → Upload defaults)
────────────────────────────────────────────────────
Step 6:      Upload Pilot #10 (private/unlisted) ← Wait for publish approval
Step 7:      Set Pilot #10 as channel trailer    (YouTube Studio → Branding)
Step 8:      Create "Start Here" playlist        (YouTube Studio → Playlists)
────────────────────────────────────────────────────
Step 9:      Final review + publish
```

**Estimated time for Steps 1–5: ~20 minutes**

## Risk Check

| Risk | Mitigation |
|------|-----------|
| Channel goes live before branding is updated | Complete Steps 1–5 before uploading any video |
| First viewers see inconsistent branding | Profile + banner must be updated first |
| Comments arrive before moderation is understood | Know the policy before video goes public |
| AI content appears without disclosure | Disclosure statement in About section before any AI video |
