# Production SOPs

> Standard Operating Procedures for the NDB-TV production pipeline.

---

## SOP-001: Research

**Purpose:** Gather reliable, relevant information for content creation.

### Process

1. **Define scope** — What question does this content answer?
2. **Primary sources** — Official docs, expert interviews, original data
3. **Secondary sources** — Reputable media, academic papers, recognized authorities
4. **Local relevance check** — How does this apply to Nouadhibou?
5. **Language check** — Can key concepts be explained in Arabic/Hassaniya?
6. **Source archive** — Save links and references for show notes
7. **Fact-check** — Verify all claims against at least 2 sources

### Time Allocation
- 5-min episode: 2–4 hours research
- 15-min episode: 4–8 hours research
- Livestream topic: 1–2 hours prep

## SOP-002: Scripting

**Purpose:** Write clear, engaging, accurate scripts.

### Process

1. **Outline** — Main points in logical order
2. **Hook** — First 15 seconds (write this first)
3. **Draft** — Write in conversational Arabic/Hassaniya
4. **Simplify** — Remove jargon, explain what remains
5. **Time check** — Read aloud, target 120–150 words/min
6. **Review** — Check for accuracy, tone, and mission alignment
7. **Final** — Add visual cues for production

### Script Format

```
[HOOK - 0:00-0:15]
(Hook text here — question, surprising fact, direct address)

[INTRO - 0:15-1:00]
(Host intro, topic framing)

[CONTENT BLOCK 1 - 1:00-4:00]
- Point A
- Point B
- Visual: [description of visual/graphic needed]

[CONTENT BLOCK 2 - 4:00-7:00]
...

[SUMMARY - last 60s]
Key takeaways

[CTA - last 15s]
Subscribe, comment, next episode teaser
```

## SOP-003: Filming

**Purpose:** Capture high-quality footage efficiently.

### Pre-Production
- [ ] Script finalized
- [ ] Location set
- [ ] Lighting tested
- [ ] Audio tested (clap test)
- [ ] Camera settings confirmed
- [ ] Teleprompter ready (if used)
- [ ] Backup media ready

### During Production
- Record 5 seconds of room tone before each segment
- Clap at start of each take for sync
- Record 10+ seconds of B-roll per scene
- Record backup audio (separate recorder if available)
- Shoot for the edit — hold each position 5 seconds

### Post-Production Handoff
- Label files: YYYY-MM-DD_Program_Shot-Number
- Transfer footage immediately
- Log best takes
- Note any audio issues

## SOP-004: Editing

**Purpose:** Assemble raw footage into a polished episode.

### Workflow
1. **Rough cut** — Assembly, no effects
2. **Audio pass** — Clean audio, leveling, background music
3. **Visual pass** — B-roll, graphics, color correction
4. **Text pass** — Lower thirds, captions, overlays
5. **Review pass** — Watch through completely
6. **Export** — 1080p at minimum
7. **Quality check** — Render review before upload

### Quality Gates
- No audio clipping
- Consistent volume throughout
- Jump cuts removed (or smoothed with B-roll)
- Graphics correctly spelled
- Colors consistent

## SOP-005: Publishing

**Purpose:** Release content consistently and correctly.

### Pre-Publish Checklist
- [ ] Metadata ready (title, description, tags, chapters)
- [ ] Thumbnail uploaded
- [ ] Subtitles ready (Arabic, English if applicable)
- [ ] End screen elements set
- [ ] Social media posts drafted
- [ ] Community post drafted

### Publishing Schedule
- Core episodes: Same day/time each week
- Shorts: Between core episodes
- Livestreams: Scheduled 7 days in advance

### Post-Publish
- [ ] Social media cross-post within 30 minutes
- [ ] Community post within 1 hour
- [ ] Monitor comments in first 24 hours
- [ ] Track analytics weekly
- [ ] Respond to community questions

## SOP-006: Shorts Extraction

**Purpose:** Create effective short-form content from full episodes.

### Selection Criteria
Best moments for shorts:
- A single, complete point (standalone)
- High emotional impact (surprise, insight, inspiration)
- Visual interest (demonstration, before/after)
- Clear call to action implicit

### Production
1. Identify 2–3 candidate moments per episode
2. Edit to 30–60 seconds
3. Tight vertical crop (9:16)
4. Add bold text overlay
5. Optimize hook for first 2 seconds
6. Separate title from main video
7. Publish minimum 24h after main video
