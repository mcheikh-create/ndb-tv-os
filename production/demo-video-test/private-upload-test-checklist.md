# NDB-TV — Private Upload Test Checklist

**For Claude Chrome (or founder):** Use this checklist when uploading the demo video to YouTube.

---

## Pre-Upload

| # | Item | Check |
|---|------|-------|
| 1 | Demo video file exported and verified (plays correctly locally) | ☐ |
| 2 | Demo thumbnail PNG exported | ☐ |
| 3 | Demo captions SRT file created | ☐ |
| 4 | Title includes "PRIVATE TEST" | ☐ |

---

## Upload Settings

| # | Field | Value | Check |
|---|-------|-------|-------|
| 5 | **Title** | NDB-TV — اختبار خاص (Private Demo Test) | ☐ |
| 6 | **Description** | Internal test only. Not a public video. This is a private workflow validation for NDB-TV. لا يُنشر للجمهور | ☐ |
| 7 | **Visibility** | **Private** or **Unlisted** (founder preference) | ☐ |
| 8 | **Category** | Education | ☐ |
| 9 | **Thumbnail** | Upload `ndb-tv-private-demo-thumbnail.png` | ☐ |
| 10 | **Captions (Arabic)** | Upload `ndb-tv-private-demo-captions.srt` | ☐ |
| 11 | **Captions (French)** | Upload French SRT if created | ☐ |
| 12 | **Playlist** | Do NOT add to any public playlist | ☐ |
| 13 | **Comments** | Off or limited to founder only | ☐ |
| 14 | **Age restriction** | No | ☐ |
| 15 | **Made for kids** | No | ☐ |
| 16 | **Allow embedding** | No (keep internal) | ☐ |
| 17 | **Paid promotion** | No | ☐ |

---

## Post-Upload

| # | Item | Check |
|---|------|-------|
| 18 | Video visible on channel (but not public) | ☐ |
| 19 | Thumbnail displays correctly | ☐ |
| 20 | Captions appear when CC is turned on | ☐ |
| 21 | Description renders correctly | ☐ |
| 22 | Copy private link and share with founder for review | ☐ |
| 23 | Note: Video can be deleted after review is complete | ☐ |

---

## Do NOT

| ❌ Action | Reason |
|-----------|--------|
| Set visibility to Public | Demo is not for public consumption |
| Add to launch playlist | Would confuse with real content |
| Use as channel trailer | Channel is empty — no trailer needed |
| Share link publicly | Internal test only |
| Keep video forever | Should be deleted or made private after review |
