# Weekly Production

## Purpose

Consistent, predictable weekly production schedule.

---

## Weekly Schedule Template

| Day | Activity | Owner | Deliverable |
|-----|----------|-------|-------------|
| Saturday | Topic selection, research kickoff | Producer | Topic brief |
| Sunday | Research complete, script draft | Producer/Writer | Script v1 |
| Monday | Script review, revisions | Editor | Script final |
| Tuesday | Filming day | Producer/Crew | Raw footage |
| Wednesday | Editing, thumbnail design | Editor | Rough cut |
| Thursday | Review, revisions, final cut | Team | Final video |
| Friday | Publishing, social promotion | Producer | Published episode |
| Saturday | Livestream (weekly) | Producer | Live broadcast |

**Note:** Adjust based on team availability. The key is consistency — same day each week.

---

## Weekly Meeting

**When:** Monday, 30 min
**Attendees:** Production team
**Agenda:**
1. Last week's episode review (metrics, feedback)
2. This week's episode status
3. Next week's topic preview
4. Blockers and needs
5. Short-form content plan

## Production Capacity

| Team Size | Weekly Output |
|-----------|---------------|
| 1 person | 1 core video + 2 shorts |
| 2 people | 2 core videos + 4 shorts |
| 3+ people | 2 core videos + 1 livestream + shorts |
