# NDB-TV Workflows — Production Directory

## Purpose
Standard operating procedures for every stage of NDB-TV content production.

## Workflow Documents

| File | Covers |
|------|--------|
| `episode-production-workflow.md` | End-to-end episode creation from concept to publish |
| `avatar-production-workflow.md` | AI avatar video production pipeline |
| `education-series-workflow.md` | Multi-episode educational series planning |
| `review-and-approval-checklist.md` | Quality control before any content goes live |

## Workflow Principles
1. **Every episode follows the same process.** Consistency ensures quality.
2. **Review before publish.** No content goes live without founder/editor approval.
3. **Document everything.** Each episode should be reproducible.
4. **Continuous improvement.** Review workflows quarterly and update based on experience.
